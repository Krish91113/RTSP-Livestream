# RTSP Livestream Overlay System

A real-time video overlay management system enabling users to place text and image overlays on video streams.

## Features
- **Live Video Playback:** Supports MP4, HLS, and RTSP streams.
- **Overlay Management:** Create, edit, and delete text/image overlays.
- **Interactive Editor:** Drag and drop overlays directly on the video player.
- **Real-time Updates:** Changes are immediately reflected on the stream view.

## Technologies
- **Frontend:** React, Vite, TypeScript, Tailwind CSS
- **Backend:** Flask, MongoDB
- **Video:** HLS.js for streaming support

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```

2. Start the development server:
   ```sh
   npm run dev
   ```
